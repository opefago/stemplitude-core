## 💡 Customization Tips

### Change Colors
Edit `src/App.css` `:root` variables

### Update Content
Edit data arrays in respective `.jsx` files

### Add Images
Place in `/public` folder and reference as `/image.jpg`

### Modify Animations
Adjust Framer Motion props in components

### Change Fonts
Update Google Fonts import in `src/index.css`
