# Stemplitude Landing Page

Standalone Next.js + Tailwind landing page for `stemplitude.com`.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy on Vercel

1. Import this repository in Vercel.
2. Set the **Root Directory** to `landing-page`.
3. Framework preset should auto-detect as **Next.js**.
4. Click **Deploy**.

No backend services are required for this project.
